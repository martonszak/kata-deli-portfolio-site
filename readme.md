## Preview/Install
You can follow the development and view the site on [this][Project-url] link.

If you download the repository:  
- ```git clone```
- open the project in VSC *(or what you prefer)* than run ```npm install```
- ```npm build```
- ```npm preview```
- open **localhost:4173**
